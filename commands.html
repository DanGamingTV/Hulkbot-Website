Hulkbot Commands


Owner Only:


h!changegame -
Changes the bot's playing status.


h!shutdown -
A fancy way to crash the bot.


h!restart -

Admin Only


h!mute -
Mutes people. Usage: h!mute @user 20s reason


h!kick -
Kicks a user. Usage: h!kick @user

h!ban -
Bans a user. Usage: h!ban @user




Everyone

h!auth -
Authorizes a user. Usage: h!auth


h!ping -
Returns the bot's ping. Usage: h!ping


h!uptime -
Returns the bot's uptime. Usage: h!uptime


h!say -
Makes the bot say what you tell it to. Usage: h!say I Like Turtles
